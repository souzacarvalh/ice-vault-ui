# IceVault

Welcome to IceVault

The IceVault is a safe place where you can keep your sensitive information.

## How do I run it?

Execute 'ng serve' it will start up a dev server on http://localhost:4200

## Dev Requirements

NodeJS (npm) and ng (Angular-CLI)
